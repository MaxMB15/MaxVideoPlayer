## Default Permission

Allows all MPV player commands

#### This default permission set includes the following:

- `allow-mpv-load`
- `allow-mpv-play`
- `allow-mpv-pause`
- `allow-mpv-stop`
- `allow-mpv-seek`
- `allow-mpv-set-volume`
- `allow-mpv-set-bounds`
- `allow-mpv-set-visible`
- `allow-mpv-get-state`
